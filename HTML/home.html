Example

  <!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
  </html>
Try it Yourself

Example Explained

  • <!DOCTYPE html> declares this as an HTML5 document
  • <html> is the root element of the page
  • <head> contains metadata about the page
  • <body> holds the visible content
