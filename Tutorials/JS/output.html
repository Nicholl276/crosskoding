JavaScript Display Options

JavaScript can present data in various manners:

  • By writing into an HTML element using innerHTML or innerText.
  • By outputting directly to the HTML using document.write().
  • By displaying data in an alert box through window.alert().
  • By logging information in the browser console using console.log().

Using innerHTML

To access an HTML element, you can use the document.getElementById(id) method.

Assign an id attribute to the desired HTML element.

Then, modify the element's HTML content using the innerHTML property.

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My First Paragraph</p>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = "<h2>Hello World</h2>";
</script>

</body>
</html>
Try it Yourself

Using innerText

To access an HTML element, use the document.getElementById(id) method.

Then use the innerText property to change the inner text of the HTML element:

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My First Paragraph</p>

<p id="demo"></p>

<script>
document.getElementById("demo").innerText = "Hello World";
</script>

</body>
</html>

Using document.write()

For testing purposes, it is convenient to use document.write():

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My first paragraph.</p>

<script>
document.write(5 + 6);
</script>

</body>
</html>

JavaScript Print

JavaScript does not have any print object or print methods.

You cannot access output devices from JavaScript.

The only exception is that you can call the window.print() method in the browser to print the content of the current window.

Example

<!DOCTYPE html>
<html>
<body>

<button onclick="window.print()">Print this page</button>

</body>
</html>

External References

There are three ways to reference an external script :

  • By using a full URL (i.e., a complete web address)
  • By specifying a file path (such as /js/)
  • Without including any path

This example utilizes a full URL to link to myScript.js:

Example

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
Try it Yourself

This example uses a file path to link to myScript.js:

Example

<script src="/js/myScript.js"></script>
Try it Yourself

This example uses no path to link to myScript.js:

Example

<script src="myScript.js"></script>
