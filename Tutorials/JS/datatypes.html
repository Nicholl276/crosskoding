JavaScript Data Types

Previous Next

JavaScript supports 8 data types:

  • String
  • Number
  • Bigint
  • Boolean
  • undefined
  • Null
  • Symbol
  • Object

The Object Data Type

The object data type can include both built-in objects and those defined by the user. Built-in types include objects, arrays, dates, maps, sets, integer arrays, float arrays, promises, and more.

Example

// Numbers:
let length = 16;
let weight = 7.5;

// Strings:
let color = "Yellow";
let lastName = "Johnson";

// Booleans
let x = true;
let y = false;

// Object:
const person = {firstName:"John", lastName:"Doe"};

// Array object:
const cars = ["Saab", "Volvo", "BMW"];

// Date object:
const date = new Date("2022-03-25");
Try it Yourself

Understanding Data Types

Data types are a fundamental concept in programming.

To effectively work with variables, it's essential to understand their types.

Without them, a computer wouldn't be able to reliably compute operations such as:

Example

let x = 16 + "Volvo";
Try it Yourself

Does it make sense to add "Volvo" to sixteen? Would that trigger an error or yield a meaningful result?

Example

let x = "16" + "Volvo";
Try it Yourself

Example

let x = 16 + "Volvo";
Try it Yourself

Example

let x = "Volvo" + 16;
Try it Yourself

JavaScript processes such examples by evaluating expressions from left to right, meaning different sequences can yield different outcomes.

Javascript

let x = 16 + 4 + "Volvo";
// result 20Volvo
Try it Yourself

Javascript

let x = "Volvo" + 16 + 4;
// result Volvo164
Try it Yourself

In one case, JavaScript treats 16 and 4 as numbers until it encounters "Volvo." In another, because the first operand is a string, all subsequent operands are handled as strings.


Dynamic Types in JavaScript

JavaScript is dynamically typed, which means a single variable can hold values of different data types over its lifetime.

Javascript

let x;       // Now x is undefined
x = 5;       // Now x is a Number
x = "John";  // Now x is a String
Try it Yourself

JavaScript Strings

A string, or text string, is a sequence of characters such as "John Doe."

Strings are defined by enclosing characters within single or double quotes.

Example

// Using double quotes:
let carName1 = "Volvo XC60";

// Using single quotes:
let carName2 = 'Volvo XC60';
Try it Yourself

You can also include quotes inside a string provided they differ from the ones that enclose the string.

Example

// Single quote inside double quotes:
let answer1 = "It's alright";

// Single quotes inside double quotes:
let answer2 = "He is called 'Johnny'";

// Double quotes inside single quotes:
let answer3 = 'He is called "Johnny"';
Try it Yourself

JavaScript Numbers

All numbers in JavaScript are stored as floating-point decimals.

They can be written with or without a decimal point.

Example

// With decimals:
let x1 = 34.00;

// Without decimals:
let x2 = 34;
Try it Yourself

Exponential Notation

Numbers that are very large or very small can be represented using scientific (exponential) notation.

Example

 let y = 123e5;    // 12300000
let z = 123e-5;   // 0.00123
Try it Yourself

JavaScript BigInt

While typical JavaScript numbers use a 64-bit floating-point format, BigInt is a newer data type (introduced in ES2020) that allows storage of integer values too large to be represented by standard JavaScript numbers.

Example

let x = BigInt("123456789012345678901234567890");
Try it Yourself

JavaScript Booleans

Booleans can have only two values: true or false.

Example

 let x = 5;
let y = 5;
let z = 6;
(x == y)       // Returns true
(x == z)       // Returns false
Try it Yourself

They are commonly used in conditional testing.


JavaScript Arrays

Arrays are defined using square brackets, with items separated by commas.

For example, the code below creates an array called cars containing three car names.

Example

const cars = ["Saab", "Volvo", "BMW"];
Try it Yourself

Remember, array indexes start at 0, so the first item is at index [0], the second at [1], and so on.


JavaScript Objects

Objects are created using curly braces {}.

Their properties are specified as key-value pairs, separated by commas.

Example

const person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
Try it Yourself

For instance, a person object might have properties such as firstName, lastName, age, and eyeColor.


The typeof Operator

The typeof operator allows you to determine the type of a JavaScript variable or expression.

Example

typeof ""             // Returns "string"
typeof "John"         // Returns "string"
typeof "John Doe"     // Returns "string"
Try it Yourself

Example

typeof 0              // Returns "number"
typeof 314            // Returns "number"
typeof 3.14           // Returns "number"
typeof (3)            // Returns "number"
typeof (3 + 4)        // Returns "number"
Try it Yourself

Undefined

In JavaScript, if a variable is declared without an assigned value, it defaults to undefined, and its type is undefined as well.

Example

 let car;    // Value is undefined, type is undefined
Try it Yourself

You can clear any variable by setting its value to undefined, which also resets its type.

Example

car = undefined;    // Value is undefined, type is undefined
Try it Yourself

Empty Values

An empty value is different from undefined. For example, an empty string has a defined value and type, even though it contains no characters.

Example

 let car = "";    // The value is "", the typeof is "string"
Try it Yourself
