The <script> Tag

In HTML, JavaScript code is embedded between the <script> and </script> tags.

Example

<script>
    document.getElementById("demo").innerHTML = "My First JavaScript";
</script>
Try it Yourself

JavaScript Functions and Events

A JavaScript function is a segment of code that executes when it is invoked.

For instance, a function may run in response to an event, such as when a user clicks a button.


JavaScript in the <head> or <body>

An HTML document can include as many scripts as desired.

Scripts can be embedded within the <head> section, the <body> section, or even in both.


JavaScript in <head>

This example shows a JavaScript function embedded in the <head>section of an HTML document.

The function is executed when a button is clicked:

Example

<!DOCTYPE html>
<html>
<head>
<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>
<body>

<h2>Demo JavaScript in Head</h2>

<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>

</body>
</html>
Try it Yourself

JavaScript in <body>

In this example, a JavaScript function is positioned within the <body> section of an HTML page.

The function is triggered when a button is clicked:

Example

<!DOCTYPE html>
<html>
<body>

<h2>Demo JavaScript in Body</h2>

<p id="demo">A Paragraph</p>

<button type="button" onclick="myFunction()">Try it</button>

<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>

</body>
</html>
Try it Yourself

External JavaScript

Scripts can also be placed in external files :

Example

function myFunction() {
  document.getElementById("demo").innerHTML = "Paragraph changed.";
}
Try it Yourself

External scripts are useful when the same code is deployed across various web pages.

JavaScript files use the .js file extension.

To include an external script, specify the script fileâ€™s name in the src (source) attribute of a <script> tag:

Example

<script src="myScript.js"></script>
Try it Yourself

You can include an external script reference in either the <head> or <body> section, based on your preference.

The script will function as if it were directly placed at the position of the <script> tag.


External JavaScript Benefits

Storing scripts in separate files offers several advantages :

  • It decouples HTML from the JavaScript code.
  • It enhances the readability and maintainability of both HTML and JavaScript.
  • Cached external JavaScript files can improve page load performance.

To include multiple script files on a single page, simply use several script tags :


JavaScript Can Modify HTML Attribute Values

In this example, JavaScript updates the src (source) attribute of an <img> tag.

Example

<script src="myScript1.js"></script>
<script src="myScript2.js"></script>

External References

There are three ways to reference an external script :

  • By using a full URL (i.e., a complete web address)
  • By specifying a file path (such as /js/)
  • Without including any path

This example utilizes a full URL to link to myScript.js:

Example

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
Try it Yourself

This example uses a file path to link to myScript.js:

Example

<script src="/js/myScript.js"></script>
Try it Yourself

This example uses no path to link to myScript.js:

Example

<script src="myScript.js"></script>
