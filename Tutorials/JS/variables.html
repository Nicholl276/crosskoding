JavaScript variables

Previous Next

Variables as Data Containers

JavaScript variables act as containers for storing data and can be declared in four different ways:

  • Automatically
  • Using var
  • Using let
  • Using const

In the initial example, x, y, and z are used without prior declaration.

They are automatically declared when first used:

Example

x = 5;
y = 6;
z = x + y;
Try it Yourself

From the examples, you can deduce that:

  • x holds the value 5
  • y holds the value 6
  • z holds the value 11

Example using var

var x = 5;
var y = 6;
var z = x + y;
Try it Yourself

Example using let

let x = 5;
let y = 6;
let z = x + y;
Try it Yourself

Example using const

const x = 5;
const y = 6;
const z = x + y;
Try it Yourself

Mixed Example

const price1 = 5;
const price2 = 6;
let total = price1 + price2;
Try it Yourself

The variables price1 and price2 are declared with the const keyword, meaning their values cannot be changed.

In contrast, the variable total is declared with let, which allows its value to be modified.


Just Like Algebra

Just as in algebra, variables store values and can be used in expressions.

Example

let x = 5;
let y = 6;
Try it Yourself

Example

let z = x + y;
Try it Yourself

From the example above, you can infer that the total is computed as 11.


JavaScript Identifiers

Every JavaScript variable must have a unique name, known as an identifier. Identifiers can be brief (like x and y) or more descriptive (such as age, sum, or totalVolume).

The general rules for creating variable names (unique identifiers) are:

  • Names may include letters, digits, underscores, and dollar signs.
  • Names must start with a letter.
  • Names can also begin with $ or _ (although this tutorial will not focus on these).
  • Names are case sensitive (for example, y and Y are considered different).
  • Reserved words, such as JavaScript keywords, cannot be used as identifiers.

The Assignment Operator

In JavaScript, the equal sign (=) serves as an assignment operator rather than an equality test operator.

Example

x = x + 5
Try it Yourself

Unlike algebra, the statement x = x + 5 is perfectly valid in JavaScript: it calculates x + 5 and assigns the resulting value back to x, effectively increasing x by 5.


JavaScript Data Types

JavaScript variables can store numbers (like 100) and text values (such as "John Doe"). In programming, text values are known as strings

Although JavaScript supports various data types, for now, consider only numbers and strings.

Strings must be enclosed in single or double quotes, while numbers are written without quotes.

If you enclose a number in quotes, it is treated as a string.

Example

const pi = 3.14;
let person = "John Doe";
let answer = 'Yes I am!';
Try it Yourself

Declaring a JavaScript Variable

Declaring a variable in JavaScript means creating it.

You can declare a variable using the var or let keyword.

Example

var carName;
or:
let carName;
Try it Yourself

Once declared, the variable has no assigned value (technically, it is undefined).

To assign a value, use the equal sign.

Example

carName = "Volvo";
Try it Yourself

You can also assign a value during the declaration.

Example

let carName = "Volvo";
Try it Yourself

For example, the code below creates a variable named carName and assigns it the value "Volvo".

The value is then output inside an HTML paragraph with the id "demo".

Example

<p id="demo"></p> 

<script>
let carName = "Volvo";
document.getElementById("demo").innerHTML = carName;
</script>
Try it Yourself

One Statement, Many Variables

You can declare multiple variables in a single statement by starting with let and separating each variable with a comma.

Example

let person = "John Doe", carName = "Volvo", price = 200;
Try it Yourself

Such declarations can extend over several lines if needed.

Example

let person = "John Doe",
carName = "Volvo",
price = 200;
Try it Yourself

Value = undefined

Often, variables in programs are declared without an initial value, as the value might be computed later or provided via user input.

A variable declared without a value is automatically assigned the value undefined.

For instance, if carName is declared without an assignment, it will hold undefined.

Example

let carName;
Try it Yourself

Re-declaring JavaScript Variables

If you re-declare a variable using var, its current value remains unchanged.

For example, re-declaring carName will not alter its existing value of "Volvo".

Example

var carName = "Volvo";
var carName;
Try it Yourself

JavaScript Arithmetic

Similar to algebra, you can perform arithmetic operations with JavaScript variables using operators like = and +.

Example

let x = 5 + 2 + 3;
Try it Yourself

Additionally, you can add strings together; however, string addition results in concatenation.

Example

let x = "John" + " " + "Doe";
Try it Yourself

Experiment with different operations to see how they behave.

Example

let x = "5" + 2 + 3;
Try it Yourself

Example

let x = 2 + 3 + "5";
Try it Yourself

JavaScript Dollar Sign $

Since JavaScript treats the dollar sign as a letter, variable names can include $

Example

let $ = "Hello World";
let $$$ = 2;
let $myMoney = 5;
Try it Yourself

Although not very common, professional developers sometimes use $ as a shorthand for the main function in a JavaScript library.

For example, in jQuery, the $ function is used to select HTML elements; $("p") selects all paragraph elements.


JavaScript Underscore (_)

Likewise, JavaScript treats the underscore as a letter, so identifiers can include _.

let _lastName = "Johnson";
let _x = 2;
let _100 = 5;
Try it Yourself

While its usage is not widespread, a common convention among developers is to use _ as a prefix to indicate that a variable is intended to be private (hidden).
