JavaScript Functions

Previous Next

A JavaScript function is a segment of code crafted to perform a specific task.

It executes when something calls (or invokes) it.

Example

// Function to compute the product of p1 and p2
function myFunction(p1, p2) {
  return p1 * p2;
}
Try it Yourself

JavaScript Function Syntax

A JavaScript function is declared using the function keyword, followed by a name and parentheses ().

Function names can include letters, digits, underscores, and dollar signs (following the same rules as variable names).

The parentheses may contain parameter names separated by commas, like so: (parameter1, parameter2, ...)

The code that the function runs is enclosed within curly brackets {}.

Example

function name(parameter1, parameter2, parameter3) {
  // code to be executed
}
Try it Yourself

Function parameters are defined inside the parentheses in the function declaration, while function arguments are the actual values passed to the function when it is called.

Inside the function, these parameters behave as local variables.


Function Invocation

The function's code is executed when something triggers it, such as:

  • When an event occurs (e.g., a user clicks a button)
  • When it's called from within other JavaScript code
  • Automatically (self-invoked)

You'll learn more about how functions are invoked later in this tutorial.


Function Return

When JavaScript encounters a return statement, the function stops executing immediately.

If the function was called from a statement, control is returned to the code following that statement.

Functions typically compute a value, which is then returned to the caller.

Example

Calculate the product of two numbers, and return the result:

// Function is called, the return value will end up in x
let x = myFunction(4, 3);

function myFunction(a, b) {
// Function returns the product of a and b
  return a * b;
}
Try it Yourself

The () Operator

The () operator is used to invoke (call) the function.

Example

Convert Fahrenheit to Celsius:

function toCelsius(fahrenheit) {
  return (5/9) * (fahrenheit-32);
}

let value = toCelsius(77);
Try it Yourself

Providing incorrect parameters when calling a function might lead to an incorrect result, while accessing a function without using () returns the function itself rather than its output.

Example

Convert Fahrenheit to Celsius:

function toCelsius(fahrenheit) {
  return (5/9) * (fahrenheit-32);
}

let value = toCelsius;
Try it Yourself

Functions as Variable Values

Functions can be assigned to variables and used in expressions, assignments, and calculations just like any other variable.

Example

Instead of using a variable to store the return value of a function:

let x = toCelsius(77);
let text = "The temperature is " + x + " Celsius";

You can use the function directly, as a variable value:

let text = "The temperature is " + toCelsius(77) + " Celsius";
Try it Yourself

Local Variables

Variables declared within a JavaScript function are local to that function and can only be accessed from within it.

Example

// code here can NOT use carName

function myFunction() {
  let carName = "Volvo";
  // code here CAN use carName
}

// code here can NOT use carName
Try it Yourself

Since local variables are confined to their respective functions, the same name can be reused in different functions without conflict.

These local variables are created when the function starts and are removed once the function finishes executing.
