JavaScript Programs

A computer program is a sequence of instructions that a computer executes.

In any programming language, these instructions are known as statements.

Thus, a JavaScript program is simply a collection of such statements.


JavaScript Statements

JavaScript statements are made up of:

  • Values
  • Operators
  • Expressions
  • Keywords
  • Comments

For example, the following statement directs the browser to insert "Hello Dolly." into an HTML element with the id "demo":

Example

document.getElementById("demo").innerHTML = "Hello Dolly.";
Try it Yourself

Typically, JavaScript programs include many statements that run sequentially in the order they appear.


Semicolons ;

Semicolons are used to delimit JavaScript statements.

It is important to place a semicolon at the end of every executable statement.

Example

let a, b, c;  // Declare 3 variables
a = 5;        // Assign the value 5 to a
b = 6;        // Assign the value 6 to b
c = a + b;    // Assign the sum of a and b to c
Try it Yourself

When using semicolons, you can even write several statements on the same line.

Example

a = 5; b = 6; c = a + b;
Try it Yourself

JavaScript White Space

JavaScript ignores extra spaces, which means you can use white space to make your code more legible.

The code examples below yield the same outcome:

Example

let person = "Hege";
let person="Hege";
Try it Yourself

It is also a good practice to include spaces around operators (such as =, +, -, *, /).

Example

let x = y + z;
Try it Yourself

JavaScript Line Length and Line Breaks

For improved readability, developers often avoid having code lines that exceed 80 characters.

If a JavaScript statement is too long for one line, the best break point is immediately after an operator.

Example

document.getElementById("demo").innerHTML =
"Hello Dolly!";
Try it Yourself

JavaScript Code Blocks

JavaScript statements can be organized into blocks using curly braces { ... }.

These code blocks group statements together so they execute as one unit.

One common example of such grouping is found in JavaScript functions.

Example

function myFunction() {
  document.getElementById("demo1").innerHTML = "Hello Dolly!";
  document.getElementById("demo2").innerHTML = "How are you?";
}
Try it Yourself

JavaScript Keywords

JavaScript statements typically begin with a keyword that indicates the action to be performed.

Our Reserved Words Reference provides a complete list of JavaScript keywords.

Below is a selection of some keywords you will encounter in this tutorial.

Keyword Description
var Declares a variable
let Declares a block variable
const Declares a block constant
if Marks a block of statements to be executed on a condition
switch Marks a block of statements to be executed in different cases
for Marks a block of statements to be executed in a loop
function Declares a function
return Exits a function
try Implements error handling to a block of statements
