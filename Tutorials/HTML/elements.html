JavaScript Introduction

Previous Next

What exactly is JavaScript?

JavaScript serves as the primary programming language for the web.

It is capable of modifying and transforming both HTML and CSS.

Additionally, it can compute, adjust, and verify data.


Why Learn JavaScript?

Every web developer must master three key languages:

  • HTML to structure the content of web pages.
  • CSS to design and layout web pages.
  • JavaScript to control the interactive behavior of web pages.

JavaScript Can Modify HTML Content

Among the various HTML methods available in JavaScript is getElementById().

In the example below, an HTML element (with id="demo") is located, and its content (innerHTML) is updated to "Hello JavaScript":

Example

document.getElementById("demo").innerHTML = "Hello JavaScript";
Try it Yourself

Example

document.getElementById('demo').innerHTML = 'Hello JavaScript';
Try it Yourself

JavaScript Can Modify HTML Attribute Values

In this example, JavaScript updates the src (source) attribute of an <img> tag.

Example

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Hidupkan dan Matikan Lampu</title>
  </head>
  <body>
    <h2>Contoh JavaScript: Hidupkan dan Matikan Lampu</h2>
    <p>JavaScript dapat mengubah nilai atribut HTML. Pada contoh ini, JavaScript mengubah nilai atribut <code>src</code> dari sebuah gambar.</p>

    <button onclick="document.getElementById('myImage').src='https://www.w3schools.com/js/pic_bulbon.gif'">Hidupkan Lampu</button>
    
    <img id="myImage" src="https://www.w3schools.com/js/pic_bulboff.gif" style="width:100px" alt="Lampu">
    
    <button onclick="document.getElementById('myImage').src='https://www.w3schools.com/js/pic_bulboff.gif'">Matikan Lampu</button>
  </body>
</html>
Try it Yourself

JavaScript Can Alter HTML Styles (CSS)

Modifying an HTML element's style is essentially a form of changing an HTML attribute.

Example

document.getElementById("demo").style.fontSize = "35px";
Try it Yourself

JavaScript Can Conceal HTML Elements

You can hide HTML elements simply by modifying their display property.

Example

document.getElementById("demo").style.display = "none";
Try it Yourself

JavaScript Can Show HTML Elements

You can show HTML elements simply by modifying their display property.

Example

document.getElementById("demo").style.display = "block";
